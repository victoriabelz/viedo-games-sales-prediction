# viedo-games-sales-prediction
Analysis and prediction of global video games sales using machine learning in R

## Project overview 
The project follows the CRISP-DM framework and applies regression models to predict global video game sales.

## Methodology
- Data preprocessing and feature engineering
- Model training (MLP neural network, Random Forest)
- Train / validation / test split
- Model evaluation

## Technologies
- R
- caret, randomForest
- neural networks
